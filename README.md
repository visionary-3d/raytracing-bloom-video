This is version 1 of my WebGPU raytracing + bloom implementation.
In the video below, I've explained how I created this.

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm run dev
```

### Production build

```bash
npm run build
```
